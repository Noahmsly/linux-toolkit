# Linux Toolkit

## Install

### Seperate Commands

```bash
sudo curl -L "https://github.com/d3vop-llc/linux-toolkit/releases/download/v0.0.1/linux-toolkit" -o /usr/bin/linux-toolkit
```

```bash
sudo chmod +x /usr/bin/linux-toolkit
```

```bash
linux-toolkit
```

### All in one command

```bash
sudo curl -L "https://github.com/d3vop-llc/linux-toolkit/releases/download/v0.0.1/linux-toolkit" -o /usr/bin/linux-toolkit
sudo chmod +x /usr/bin/linux-toolkit
linux-toolkit
```
